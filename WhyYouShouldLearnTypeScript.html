History of TypeScript

TypeScript is a strict super-set of JavaScript, that means you can write plain vanilla JavaScript right inside a .ts file. TypeScript was developed and is currently maintained by the fine gents at Microsoft. The original purpose of TypeScript was to allow for developers to create large scale JavaScript applications with type safety. Unlike other languages such as Script#, GWT and Dart, TypeScript doesn't ditch JavaScript to the curb and implement its own syntax on it's own. Remember, JavaScript works as it should, 100% of the time alongside TypeScript code.

Don't worry, TypeScript didn't just appear last Wednesday, it's been around since 2012 with two years of development behind the scenes at Microsoft. TypeScript is transpiled to JavaScript under numerous modules(AMD, UMD, CommonJs) and different target specificiations(ES3, ES4, ES5, ES6, etc)



Transpiled?

This is the beauty of TypeScript. It is transpiled!! Basically a transpilation process is nothing more than a source-to-source compilation. You take one piece of source code in one language and convert it to another language. TypeScript -> JavaScript. The transpiled code to JavaScript is idiomatic JavaScript, it will be simple to read and understand if you're a JavaScript developer.

The beauty here is that the TypeScript language can keep getting developed to suit the needs of developers while Browsers adopt whatever the current JavaScript specification is such as ES5 or ES6. The developers can start using new features such as classes and interfaces before they're rolled out to all the major browsers(ES6)! But Oleg, when ES6 rolls out, TypeScript is pointless! False! There are a ton of features that are coming with ES6 and it will take a long while for browsers to adopt them all, years. Why would you wait years when you can start using them today? Why not with one single line of code transpile to ES6 once its production ready?

    Greeter.prototype.greet = function () {
        return "Hello, " + this.greeting;
    };
    return Greeter;
})();

Inheritance


//TypeScript
class Animal {
    constructor(public name: string) { }
    move(meters: number) {
        alert(this.name + " moved " + meters + "m.");
    }
}

class Snake extends Animal {
    constructor(name: string) { super(name); }
    move() {
        alert("Slithering...");
        super.move(5);
    }
}

class Horse extends Animal {
    constructor(name: string) { super(name); }
    move() {
        alert("Galloping...");
        super.move(45);
    }
}

var sam = new Snake("Sammy the Python");
var tom: Animal = new Horse("Tommy the Palomino");

sam.move();
tom.move(34);
