Node Package Manager

Probably a good idea to define the acronym to save you some Google time. TLDR: npm = NuGet...except for JavaScript libraries. It's managed by a highly skilled team at npm, Inc. Essentially npm is a huge registry of open-source, enterprise, and private packages.

NPM is a command line tool(CLI). I have seen some GUI's, but please avoid them...that just feels dirty.

How Do I Save a Package to My Solution?

Here is the magic npm install "packagename" "saveAttribute"

Notice that package.json file hanging out in the root of your solution? That json file keeps track of your 3rd party dependencies necessary to develop, build, or deploy your application.

Notice devDependencies? Notice dependencies? Let's discuss the differences!


Using -g When Installing a Package

If you want to save a package to your global node modules directory, you will append your command line input with -g.

Here is an example

    npm install bower -g

Now you can use this package in ANY project! You'll want to avoid throwing everything into your global folder because you don't want to pollute it with project specific dependencies, especially if you want to distribute your project in a lightweight manner.


Using --save When Installing a Package

Now you saved a 3rd party node package into your project folder! Typically this resides in "node_modules".

Here is an example

    npm install cors --save

Using --save-dev When Installing a Package

So you've seen people use --save-dev and you're confused. Don't worry, what this attribute does is add the package to the devDependencies section of your package.json file and signifies this as a development dependence. These are going to be packages that help in the development process but don't need to be passed around and distributed for production use.

Common dev dependencies include:

  • Task Runners such as Gulp or Grunt
  • Linting tools such as JSHint or JSLint

Here is an example

    npm install cors --save-dev

How do I quickly install all of my dependecies?

The power of npm! All you need to do is make the following call: 

    npm install

That's it! NPM will go out and install of the dependencies mentioned in your package.json file and pull them into your solution.
